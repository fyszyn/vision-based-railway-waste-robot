# vision-based-railway-waste-robot
Vision-Based Robotic System for Railway Waste Management
Overview

This project presents a prototype robotic system that detects and collects non-biodegradable waste using computer vision and robotic manipulation. The system combines a Raspberry Pi, OpenCV, YOLO object detection, and a 3D-printed robotic arm to demonstrate automated waste collection.

Features
Computer Vision using OpenCV
YOLO Object Detection
Raspberry Pi Control
Forward and Inverse Kinematics
3D Printed Robotic Arm
Servo Motor Control
Waste Pickup Prototype
